# Generator de Sfaturi (Advice Slip API)

Acest proiect este realizat pentru Tema 2. Aplicația generează sfaturi aleatorii folosind un API extern.

## API Utilizat
Am folosit **Advice Slip API** (API #8).
Link: https://api.adviceslip.com/advice

## Cum funcționează
1. Apasă pe butonul "Obține Sfat Nou".
2. Aplicația face un request către server.
3. Se afișează textul sfatului și ID-ul acestuia.

- index.html: Structura paginii
